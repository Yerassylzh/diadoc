### Клонируйте репозиторий
```bash
git clone https://github.com/Yerassylzh/diadoc
cd diadoc
```

### Создайте виртуальное окружение
#### Для macOS/Linux
```bash
python3 -m venv venv
source venv/bin/activate
```
#### Для Windows
```bash
python3 -m venv venv
venv\Scripts\activate
```

### Установите зависимости
```bash
pip install -r requirements.txt
```
